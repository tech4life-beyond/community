# Support

This is the entry-point repository for operating Tech4Life & Beyond.

## What we support here
- Onboarding and “how this works” documentation
- Contribution workflows and templates
- Organization repository map and operating flow

## How to get help
Open an issue in this repository with:
- a clear title
- what you were trying to do
- links to related repositories or PRs (if any)

## Confidentiality
Do not include private finance/legal details in public issues.

## Maintainers
Changes to core onboarding docs require CODEOWNERS review.
